# HOME-RAMEN
カップ麺や袋麺など家で食べられるラーメン専用の口コミ・投稿サービスです。

## 環境
* 言語(Ruby 2.5.8)
* フレームワーク(Ruby on Rails 5.2.4)
* データベース(PostgreSQL)
* 開発(Docker)

## 機能・技術
* デザイン(Bootstrap,Sass)
* 単体・統合テスト(RSpec,FactoryBot,Capybara)
* 静的解析ツール(RuboCop)
* ページネーション(kaminari)
* 投稿
* いいね(Ajax)
* コメント
* 通知（いいね・コメント発生時）
* 画像関連(CarrierWave)
* 検索(ransack)
